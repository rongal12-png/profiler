{% if lang == 'he' %}2. מערכת הציונים{% else %}2. Scoring System{% endif %}

{% if lang == 'he' %}כל ארנק מקבל ציון משקיע (Investor Score) בין 0-100, המורכב מ-5 רכיבים:{% else %}Each wallet receives an Investor Score between 0-100, composed of 5 components:{% endif %}

1. Balance Score (30%)

{% if lang == 'he' %}ציון לוגריתמי מבוסס על שווי נטו בדולרים:{% else %}Logarithmic score based on net worth in USD:{% endif %}

min(100, 20 * log10(max(1, net_worth_usd)))

min(100, 25 * log10(max(1, tx_count)))

5. Sybil Risk Score (-10%)

{% if lang == 'he' %}יתרה נמוכה + פעילות גבוהה = חשד ל-Sybil:{% else %}Low balance + high activity = Sybil suspicion:{% endif %}

{% if lang == 'he' %}תנאי{% else %}Condition{% endif %}{% if lang == 'he' %}ציון{% else %}Score{% endif %}

{% if lang == 'he' %}אחר{% else %}Otherwise{% endif %}0

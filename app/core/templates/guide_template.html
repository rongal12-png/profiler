  4. {% if lang == 'he' %}סיווג רמה ופרסונה:{% else %}Tier & Persona Classification:{% endif %} Whale/Tuna/Fish/Infra + 7 {% if lang == 'he' %}פרסונות{% else %}personas{% endif %}

{% if lang == 'he' %}3. סיווג רמות (Tiers){% else %}3. Tier Classification{% endif %}

{% if lang == 'he' %}רמות חלות רק על ארנקי USER. ארנקי תשתית מסווגים תמיד כ-Infra.{% else %}Tiers apply only to USER wallets. Infrastructure wallets are always classified as Infra.{% endif %}

{% if lang == 'he' %}לוגיקת הסיווג (לפי סדר עדיפות):{% else %}Classification Logic (priority order):{% endif %}

  1. {% if lang == 'he' %}ארנק תשתית (CEX/DEX/Bridge/Protocol/Contract) → Infra תמיד{% else %}Infrastructure wallet (CEX/DEX/Bridge/Protocol/Contract) → always Infra{% endif %}
  2. {% if lang == 'he' %}תווית VC / KOL / Smart Money → Whale תמיד{% else %}VC / KOL / Smart Money label → always Whale{% endif %}
  3. {% if lang == 'he' %}שווי ≥ $1M → Whale (ללא קשר לציון){% else %}Net worth ≥ $1M → Whale (regardless of score){% endif %}
  4. {% if lang == 'he' %}שווי ≥ $100K + ציון ≥ 55 → Whale, אחרת → Tuna{% else %}Net worth ≥ $100K + score ≥ 55 → Whale, otherwise → Tuna{% endif %}
  5. {% if lang == 'he' %}ציון ≥ 55 → Whale{% else %}Score ≥ 55 → Whale{% endif %}
  6. {% if lang == 'he' %}ציון 30–54 → Tuna{% else %}Score 30–54 → Tuna{% endif %}
  7. {% if lang == 'he' %}ציון < 30 → Fish{% else %}Score < 30 → Fish{% endif %}

🐋 Whale

{% if lang == 'he' %}ציון ≥ 55, או שווי ≥ $1M, או תווית VC/KOL.{% else %}Score ≥ 55, or net worth ≥ $1M, or VC/KOL label.{% endif %}

🐟 Tuna

{% if lang == 'he' %}ציון 30–54, או שווי $100K–$1M (ללא קשר לציון).{% else %}Score 30–54, or net worth $100K–$1M (regardless of score).{% endif %}

🐠 Fish

{% if lang == 'he' %}ציון < 30. משקיעים קטנים, מתחילים, או ארנקי dust.{% else %}Score < 30. Small investors, newcomers, or dust wallets.{% endif %}

⚙️ Infra ({% if lang == 'he' %}תשתית{% else %}Infrastructure{% endif %})
